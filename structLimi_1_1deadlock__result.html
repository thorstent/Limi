Detailed Description

template<class Symbol, class State>
struct Limi::deadlock_result< Symbol, State >

Represents the result of the deadlock algorithm.

If deadlock_result is false, than all other fields of this struct must be ignored (they are not set).

Template Parameters
SymbolThe type of symbols
StateThe type of states.
